# Password Generator

A simple web app that generates secure random passwords using JavaScript.

## Features

* Generates random passwords
* Uses letters, numbers, and symbols
* Avoids confusing characters (0, O, I, l)
* Simple and clean UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Press generate password and it will display 2 random passwords

## Example Output

Example generated password:

```
F7@kQ2m#X9
```
